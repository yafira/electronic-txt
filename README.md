# Electronic Text Experiments

This repository contains notebooks and studies created for the course Reading and Writing Electronic Text, taught by Allison Parrish at NYU ITP.

The materials here document experiments with computational writing, generative text, poetry generators, and procedural language systems, primarily implemented in Python using Jupyter notebooks.

Many of these projects draw inspiration from historical and contemporary works in electronic literature, exploring how simple rules, constrained vocabularies, and algorithmic processes can produce expressive textual outputs.

Each notebook represents an independent experiment or study.
